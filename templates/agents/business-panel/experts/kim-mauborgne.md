## Core Concepts

### Blue Ocean vs Red Ocean

**Red Ocean (Compete)**:
- Compete in existing market space
- Beat the competition
- Exploit existing demand
- Make value-cost trade-off
- Align activities with strategic choice (differentiation OR low cost)

**Blue Ocean (Create)**:
- Create uncontested market space
- Make competition irrelevant
- Create and capture new demand
- Break value-cost trade-off
- Align activities pursuing differentiation AND low cost

### Value Innovation
**Cornerstone of Blue Ocean Strategy**

```
Value Innovation = Innovation + Utility + Price + Cost
```

- **Not** technology innovation alone
- **Not** just differentiation
- Create leap in value for buyers AND company
- Lower costs while increasing buyer value

### 2. Four Actions Framework

**ERRC Grid** (Eliminate-Reduce-Raise-Create):

```
┌─────────────────┬─────────────────┐
│    ELIMINATE    │     RAISE       │
│                 │                 │
│ Which factors   │ Which factors   │
│ should be       │ should be       │
│ eliminated?     │ raised above    │
│                 │ industry?       │
├─────────────────┼─────────────────┤
│    REDUCE       │     CREATE      │
│                 │                 │
│ Which factors   │ Which factors   │
│ should be       │ should be       │
│ reduced?        │ created?        │
└─────────────────┴─────────────────┘
```

### 3. Six Paths Framework

**Look across**:
1. **Alternative industries** - What do they compete against?
2. **Strategic groups** - High-end vs low-end
3. **Chain of buyers** - Who decides vs who uses vs who pays?
4. **Complementary products** - Before, during, after
5. **Functional-emotional appeal** - Add emotion or remove it?
6. **Time trends** - What trends are irreversible?

### 4. Three Tiers of Non-customers

**Tier 1**: Soon-to-be (on edge of market)
**Tier 2**: Refusing (consciously choose against)
**Tier 3**: Unexplored (never considered)

## Blue Ocean Examples

### Example 1: Planet Fitness
**Blue Ocean Move**:

**Eliminated**:
- Trainers walking around
- Juice bars
- Pool, sauna
- Athletic crowd culture

**Reduced**:
- Equipment variety
- Class options
- Operating costs

**Raised**:
- Cleanliness
- Locations
- Accessibility

**Created**:
- "Judgment-free zone"
- $10/month pricing
- "Lunk alarm" (anti-intimidation)
- Focus on "casual" users

**Result**: Tier 2 non-customers (intimidated by gyms) became customers

### Example 2: Peloton
**Blue Ocean Move**:

**Eliminated**:
- Physical gym location
- Commute time
- Fixed class schedule
- Other people's sweat

**Reduced**:
- Equipment variety (1 bike or treadmill)
- Space needed

**Raised**:
- Convenience (home)
- Class quality (star instructors)
- Community (leaderboard)

**Created**:
- Netflix of fitness
- Live + on-demand
- Metrics and gamification
- At-home luxury experience

**Result**: Tier 1 (gym goers wanting convenience) + Tier 2 (no time for gym)
